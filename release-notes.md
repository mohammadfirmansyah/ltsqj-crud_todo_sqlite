## 📡 API Endpoints
- `GET /todos` - Fetch all todos
- `POST /todos` - Create a new todo
- `GET /todos/:id` - Get specific todo by ID
- `PUT /todos/:id` - Update todo by ID
- `DELETE /todos/:id` - Delete todo by ID
